# 🍽️ Food Munch

**Food Munch** is a fully responsive, stylish food delivery landing page built with HTML, CSS, and JavaScript. The design focuses on a clean layout and intuitive user experience, suitable for showcasing a modern food service.

---

## 🔗 Live Demo

👉 [View Live Site](https://foodmunch.ccbp.tech/)

---

## ✨ Features

- 🎯 Responsive design for all devices
- 🍕 Attractive UI and layout
- 🧭 Smooth navigation
- 💬 Clear call-to-actions like “Order Now”
- 💻 Built using HTML, CSS, and basic JS

---

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
